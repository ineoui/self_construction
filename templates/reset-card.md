# 状态复位卡片

适用场景：上班时空想机会、刷信息、焦虑、拖延、两边都不推进。

## 10 秒

```text
坐直
双脚踩地
肩膀放下
呼气一次
看远处 5 秒
```

## 30 秒

当前状态打勾：

```text
[ ] 维护现金流
[ ] 推进新方向
[ ] 机会焦虑
[ ] 陷入循环
[ ] 需要休息
```

给它命名：

```text
我现在是：
```

## 1 分钟

把念头放进 inbox，不展开：

```text
我想到的机会/方向：
今晚或固定窗口再处理：
```

关掉一个无关入口：

```text
我要关掉：
```

## 5 分钟

下一段只做一个可见动作：

```text
我要做：
```

最低承诺：

```text
只做 5 分钟，5 分钟后允许停。
```

完成后记录一个证据：

```text
我刚刚产出的东西是：
```
