# 从浅到深的立即行动

这份清单用在这些时刻：

- 坐在工位上，心里开始空转
- 明明该工作，却一直想外面的机会
- 方向很多，但一个都没有推进
- 感觉自己同时辜负当前工作和未来可能性

目标不是立刻变得热血，而是把状态从“脑内循环”拉回“可执行动作”。

## 第 1 层：身体复位，10 秒内

先做身体动作，不解释、不分析。

```text
坐直
双脚踩地
肩膀放下
慢慢呼气一次
眼睛看远处 5 秒
```

可替换动作：

```text
喝一口水
站起来 20 秒
去洗手间洗手
下巴稍微收回来
背从椅背上离开一点
```

这些动作的意义不是“养生”，而是给大脑一个信号：我回来了。

## 第 2 层：环境止损，1 分钟内

当发现自己开始搜索机会、比较方向、刷信息时：

```text
关掉一个无关网页
手机屏幕朝下
只留下当前任务窗口
把想到的机会写到 inbox，不展开
```

记录格式：

```text
想到：AI 工具 / 自动化 / 产品方向
处理时间：今晚 20:30
现在动作：回到当前最小交付
```

关键是：记录，但不展开。

## 第 3 层：语言命名，30 秒内

给当前状态命名，比继续在脑子里打转更有效。

```text
我现在不是在规划，我是在机会焦虑。
我现在不是没救，我只是没有下一步。
当前工作先当现金流，不向它索要意义。
外部方向先当实验，不当人生判决。
下一步不是想清楚，而是做出一个可见动作。
```

命名之后，不继续分析，只进入下一层。

## 第 4 层：任务切小，5 分钟内

不要问：

```text
我今天要不要好好工作？
```

只问：

```text
接下来 5 分钟，我能推进什么？
```

动作必须具体到动词：

```text
打开文档
写 3 行
改一个 bug
回复一条消息
整理一个需求点
看完一页资料
给方向列表打分
提交一个很小的修改
```

如果完全不想动，就用最低承诺：

```text
我只做 5 分钟，5 分钟后允许停。
```

启动比坚持更重要。

## 第 5 层：结构化脱困，半天到 14 天

更深一层的结构是：

```text
当前工作：每天只保住最低交付
外部方向：只选一个做 14 天实验
上班乱想：记到 inbox，不展开
下班学习：只做能留下证据的东西
```

判断标准不是：

```text
我有没有变得热血？
```

而是：

```text
今天有没有一个可见产出？
```

哪怕很小也算。

## 最小流程

状态差的时候只执行这一套：

```text
1. 坐直
2. 呼气
3. 写下：我现在处于什么状态？
4. 写下：下一段只做哪一个动作？
5. 做 5 分钟
```

这套流程不依赖心情，也不要求你先相信未来。
