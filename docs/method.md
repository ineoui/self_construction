# 方法：从机会焦虑回到可见行动

## 1. 先承认当前结构

这个状态通常不是单纯的懒，而是两个系统互相拉扯：

- 当前工作缺少意义、成长和上升空间
- 外部机会看起来很多，但每个都需要额外学习

于是大脑会进入一种高消耗模式：不断搜索、比较、设想转型，但很少进入真正的产出。

这个工具箱要解决的不是“如何马上找到终极方向”，而是先停止双重消耗。

## 2. 把当前工作降级成现金流项目

短期内，不再向当前工作索要意义感。

它的临时定义是：

```text
给我提供现金流，同时允许我准备下一步。
```

每天只问两个问题：

```text
今天工作上最不能炸的一件事是什么？
做到什么程度就算对得起工资？
```

这不是摆烂，而是停止把有限精力投入到已经确认低回报的意义搜索里。

## 3. 把外部方向降级成 14 天实验

不要问：

```text
我以后到底应该做什么？
```

先问：

```text
接下来 14 天，我能不能做出一个证据？
```

证据可以是：

- 一个小工具
- 一篇技术分析
- 一个作品集页面
- 一个自动化脚本
- 一个行业调研文档
- 一个可投递的简历版本

方向不是靠想清楚才开始，很多时候是靠做出反馈后逐渐变清楚。

## 4. 上班时间的处理规则

当你在上班时想到外部机会，不要立刻展开搜索。

只记录一句：

```text
我刚才想到的机会/方向是：
```

然后回到当前最小交付。

每天只安排一个固定窗口处理外部方向，比如中午 20 分钟或下班后 45 分钟。不要让机会搜索吞掉整天。

## 5. 每小时回顾问题

提醒器弹出时，只回答：

```text
0. 先坐直，双脚踩地，呼气一次。
1. 我刚才是在维护现金流，推进新方向，还是陷入机会焦虑？
2. 我有没有又进入“想很多但没有产出”的循环？
3. 下一段只推进哪一个可见动作？
```

这个动作的价值，是把模糊的失控感变成可观察记录。

## 6. 判断是否有效

14 天后不要用情绪判断，用证据判断：

- 我有没有完成一个能展示的东西？
- 我是否比 14 天前更清楚这个方向适不适合我？
- 我是否减少了上班时的机会焦虑？
- 这个方向是否值得再投入 14 天？

如果答案是否定的，就换实验，不要把它解释成“我不行”。
