# 时间回顾提醒器

这是一个本地小工具：每隔 1 小时或 2 小时弹出一次窗口，让你记录这段时间是在维护现金流、推进新方向，还是陷入机会焦虑。

## 怎么用

双击下面任意一个文件：

- `start-hourly.bat`：每 60 分钟提醒一次
- `start-two-hours.bat`：每 120 分钟提醒一次

打开后也可以在窗口里改提醒间隔，然后点“应用并重计时”。

想先测试效果，可以点窗口里的“立即回顾”，或者把间隔临时改成 5 分钟。

## 记录在哪里

记录会保存在：

```text
tools/time-checkin/logs/
```

每天会生成一个 Markdown 文件，例如 `2026-07-09.md`，同时也会写入 `entries.jsonl`，方便以后做统计或搜索。

## 建议用法

每次弹窗只写三件事：

1. 刚才实际做了什么
2. 有没有陷入想方向但不产出，触发点是什么
3. 下一段只推进哪一个可见动作

重点不是写得漂亮，而是把“我正在发生什么”从模糊状态里拉出来。
